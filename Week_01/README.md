# 课程建议

1. 建议老师重点讲解课程中的重点内容，在重点内容上多花点时间。
2. 建议课程中的提问放到课后统一回答，尽量不在课堂中解答。因为1千多人的大课，提问肯定很多，老师解答的时间会比较长。这样的问题一方面是会造成拖堂，另一方面会导致课程不太连贯。
